A simple youtube music player command-line based interface. 

**Requirements**
1. [mpv](https://mpv.io/)
2. [yt-dlp](https://snapcraft.io/install/yt-dlp/debian#install)

**Installation**
1. Git clone https://github.com/Kir0Sploit/YT-CLI.git
2. cd YT-CLI
3. chmod +x YT-CLI.sh
4. ./YT-CLI.sh [music name]
